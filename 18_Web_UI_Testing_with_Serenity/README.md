# Web UI Testing with Serenity

1. Web UI Testing bisa dilakukan menggunakan tools yang bernama Serenity. Serenity memiliki arsitektur yang berisi Requirements, Tests, Steps, Pages, Reports.
2. Format penulisan jika menggunakan Serenity adalah:

**Feature**: Feature name 
    As a ... 
    I want to ...
    So that I ...

    Scenario: Test Case Name
      Given I ... 
      When I ... 
      Then I ...

3. Serenity bisa memberikan hasil/report dari test yang telah dilakukan. Isi dari report tersebut adalah execution time, error message jika test gagal, dan masih banyak lagi.
